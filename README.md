# Making an Outdoor Living Room

This repository documents the process of creating a functional outdoor living space focused on protection from the sun and creating a cooking area. It includes practical instructions on setting up a tarp shelter and assembling a basic outdoor kitchen.

## Features

- Step-by-step guidance on rigging a tarp for shade and shelter
- Instructions for tying various knots used in outdoor setups
- Recommendations for tools and materials
- Basic outdoor kitchen setup ideas

## Tech Stack

- Markdown for documentation
- Static assets (photos) for visual reference

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/justin-napolitano/making-an-outdoor-living-room.git
   cd making-an-outdoor-living-room
   ```

2. View the documentation by opening `index.md` in a Markdown viewer or editor.

3. Browse the `photos` directory for supporting images.
